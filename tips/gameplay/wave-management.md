# Wave Management Guide

## The Three Wave States

### Freeze
- **How:** Only last-hit minions. Don't use AoE abilities on the wave.
- **When:** You're ahead, or you want to deny enemy farm
- **Why:** Keeps the wave near your tower, forcing enemy to overextend
- **Risk:** Enemy can roam while you're frozen

### Slow Push
- **How:** Kill only the ranged minions, let melee minions accumulate
- **When:** You want to roam or set up a dive
- **Why:** Builds a large wave that crashes into enemy tower
- **Result:** 30-second roam window while enemy clears the wave

### Fast Push (Shove)
- **How:** Kill all minions with abilities as fast as possible
- **When:** Enemy is dead/recalled, or you need to rotate immediately
- **Why:** Crashes wave into tower, denying enemy gold/XP, then rotate

## When to Use Each
| Situation | Wave State |
|-----------|-----------|
| You're ahead in lane | Freeze |
| You want to roam | Slow Push |
| Enemy just recalled | Fast Push + Roam |
| Objective spawning soon | Fast Push → Rotate |
| Enemy has kill pressure | Freeze near tower |

## Common Mistakes
- ❌ Auto-attacking the wave mindlessly (unintentional push)
- ❌ Freezing when your Jungler is invading nearby (rotate to help!)
- ❌ Pushing without vision of enemy Jungler
