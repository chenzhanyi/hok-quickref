# Shangguan Tips & Tricks

---

## Core Gameplay Tips

### 1. Ground Collisions > Aerial Barrage

Many beginners just want to go airborne. But **S1+S2 ink collisions and S3 dash-through-enemy damage** are her REAL burst. The aerial phase is the finisher, not the main damage.

### 2. S3 Brake (The #1 Advanced Mechanic)

During S3 dashes, **release the left joystick** to pause the dash timer (max 3 seconds). Use this to:
- Weave enhanced auto-attacks between dashes
- Wait for S1/S2 cooldowns
- Reposition
- Bait enemy Flashes before continuing the dash

### 3. Don't Start S3 from Too Far

If you dash from long range, the 10 aerial bolts get **split among Minions, tanks, and the target**. Instead: close distance first, then initiate S3 for concentrated burst on the backline.

### 4. Dash Through Enemies

S3's first 4 dashes deal massive damage when they **pass through enemy Heroes**. Don't dash around them — dash THROUGH them.

### 5. Skill Priority

**Max S1 first, then S2. Take ultimate whenever available.** S1 has more reliable line burst and better wave clear.

## Signature Combos

### Basic Long-Range Takeoff: **S2 → S3 → S3 → S1 → S3 → S3 → S3**

1. S2 behind you
2. S3 → dash twice following S2's ink return path
3. S1 toward enemy
4. S3 → dash three times following S1's ink direction
5. Airborne → 10 bolts rain down

### Instant Close-Range: **S2 → S1 → S3 → dash forward ×5**

1. S2 + S1 at point-blank range
2. S3 → push joystick straight forward 5 times
3. Instant multiple collisions + airborne
4. No time for enemy to react

### CN Server "Brake & Chase": **S2 → S3 → S3 → Brake → AA → S1 → S3 → S3 → S3**

1. S2, dash twice
2. **Release joystick (brake)** — weave enhanced AA + S2 damage
3. Wait for enemy Flash/dash
4. S1 in their escape direction → dash 3 times → airborne → finish

## Advanced Tips

- **Splendor (Stasis) is mandatory:** You're vulnerable after landing. Stasis active → sell → buy resurrection/Fate's Edge in the shop during stasis.
- **Practice in Training Mode:** Drill "2331333" until muscle memory. Then add brakes.
- **Matchup:** Destroys immobile squishies. Struggles vs instant point-and-click CC (Zhang Liang, Daji) that cancels S3 mid-dash.
