# Lian Po Tips & Tricks

---

## Core Gameplay Tips

### 1. S1 → S2 → S1: Double Knockup Combo

S2 refreshes S1 if it hits ANYTHING. The core trading pattern: **S1 knockup → S2 (hits, S1 refreshes) → S1 second knockup.** Two knockups, massive damage, Unstoppable through everything.

### 2. S3 + S1: Catch Fleeing Enemies

S3's final phase 3 is the big knockup. Enemies will Flash to dodge it. During S3 phase 1 or 2: **press S1 to dash** and reposition onto them. The phase 3 knockup now lands.

### 3. S2 Centers Enemies (Mini-Vacuum)

S2 pulls nearby enemies toward its center. Use to: group minions for faster clear, clump enemies for AoE follow-up, or pull enemies back when they try to escape.

### 4. Always Pre-Stack Battle Spirit

Battle Spirit grants 20% DR + 30% AS at max. Before fights: S1/S2 minions or jungle camps to build Spirit. Never engage at 0 Spirit.

### 5. S1 + Flash: Long-Range Engage

S1 dash → **Flash** during the dash to extend range. Surprise knockup the enemy backline from off-screen.

## Signature Combos

### Lane Trade: **S1 → S2 → S1 → AA**

1. S1 knockup
2. S2 (hits enemy + minions → S1 refreshes + max Spirit)
3. S1 second knockup
4. AA with +30% AS from max Spirit

### Long-Range Engage: **S1 + Flash → S2 → S1 → S3**

1. S1 + Flash onto backline
2. S2 (refreshes S1)
3. S1 chase
4. S3 → S1 mid-ultimate to adjust

### Maximum Control Chain: **S3 → S1 (adjust) → S3 (finish) → S2 → S1**

1. S3 phase 1+2
2. S1 to reposition if needed
3. S3 phase 3 knockup lands
4. S2 → S1 refresh → S1 second knockup

## Advanced Tips

- **All abilities are Unstoppable.** You can walk through ANY CC. Time engages when enemies waste their control.
- **S2's base damage is surprisingly high.** Even full-tank Lian Po can 1-combo squishies.
- **Matchup:** God-tier vs CC-reliant teams. Universal flex pick — works in any role.
