# Kongkong Er Tips & Tricks

---

## Core Gameplay Tips

### 1. Clone Bait: Make Enemies Waste Everything

Send a clone toward the enemy. They'll often burn CC, ultimates, or Flashes on it. Now you engage with their key cooldowns wasted. The #1 mind-game tool.

### 2. Clone → Swap: The Perfect Dive Cycle

1. Leave a clone in a safe spot (bush, behind a wall)
2. Dive the enemy backline → full combo → detonate marks
3. **Swap to the clone** → you're now safe
4. This is the "untouchable Assassin" cycle

### 3. Clone Vision: Never Face-Check

Send a clone into suspicious bushes. It provides vision. If enemies attack it, you know they're there AND they wasted skills.

### 4. Stack Marks Fully Before Detonating

Marks deal %HP Magic Damage on detonation. 3-4 marks = massive damage. 1-2 marks = mediocre. Always stack before the detonate ability.

### 5. Edge of Fight, Not Frontline

Kongkong Er is pure glass. Never be the first one in. Wait at the edge. When key enemy CC is down AND a squishy is isolated: that's your moment.

## Signature Combos

### Tower Dive Safe Kill: **Clone (outside tower) → Dash in → Marks → Detonate → Swap to clone**

1. Place clone outside tower range
2. Dash in → S1 + S2 (stack 3-4 marks)
3. Detonate (enemy dies)
4. Swap to clone → walk away

### Long-Range Ambush: **Clone forward → Swap → Marks → Detonate → Dash out**

1. Send clone toward enemy backline
2. Swap to clone (instant gap-close)
3. Stack marks → detonate
4. Dash back to safety

---

## Advanced Tips

- **Clones mirror your movements.** Skilled enemies can identify the real you by movement patterns. Act unpredictably.
- **Clone swap has a brief invulnerability frame.** Use to dodge tower shots or Zhuge Liang S3.
- **Matchup:** God-tier vs immobile squishies. Struggles vs AoE CC that hits both you and your clone.
