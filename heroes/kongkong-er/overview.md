# Kongkong Er（空空兒） — Assassin

## Overview

- **Role:** Assassin (Magic Burst / Clone Deception)
- **Lane:** Jungle / Mid Lane
- **Difficulty:** ★★★☆☆
- **Title:** 笑面詭手 (Grinning Trickster)
- **Region:** CN server (空空儿)

> Kongkong Er is a uniquely deceptive Magic Assassin. His core mechanics revolve around **clones, multi-dash mobility, and mark detonation burst.** He creates clones to confuse enemies, can swap positions with them, and stacks magic marks on targets that detonate for massive %HP burst. He's a ghost on the battlefield — appearing, deleting, and vanishing before enemies can react.

---

## Core Mechanics

### Clone Deception: Bait, Confuse, Escape

Abilities create clones. Clones can: bait enemy CC and ultimates, provide vision (send clones into bushes), and enable **position swapping** — dive, burst, swap back to safety.

### Mark Stack → Detonate

Abilities apply magic marks on enemies. AAs or specific abilities detonate the marks for **%HP Magic Damage.** Stack marks fully before detonating for maximum burst.

### Clone Swap: The Signature Outplay

When a clone is active, Kongkong Er can **swap positions with it.** Dive in → full combo → swap to the clone behind a wall → safe. Also: send clone forward → swap → instant gap-close.
